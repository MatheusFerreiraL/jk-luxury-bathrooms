## Project

Luxury bathroom renovation landing page for a Dublin-based company. Single-page React app with a separate `/projects` gallery route. No backend.

## Commands

```bash
npm run dev      # Vite dev server → http://localhost:5173
npm run build    # Production build to dist/
npm run preview  # Preview dist/ locally
npm run lint     # ESLint
```

## Tech Stack

- React 18 + Vite 5
- CSS Modules (co-located `.module.css` per component)
- React Router v6 (`BrowserRouter` in `main.jsx`)
- No backend, no database, no auth
